For this project, I hoped to accomplish provide interesting information about the Metro System, a new rail line extended east to Monrovia and light rail vehicle features. Also, I wanted to learn the following:

  • Use of Scalable Vector Graphics (.svg) format art instead of .png, .gif or .jpg;
  • Basic jQuery techniques and function creation;
  • Use of Parallax (jQuery) images that feature Metro glamour shots;
  • CSS3 Animation techniques;
  • Use GitHub repository and pages to host pages;
  • CSS Background Absolute Positioning, Background Cover;
  • Use of Metro Brand Adobe/WebKit Scala and Din webfonts;
  • HTML5 Video Background; and
  • jQuery menu swap based on <div> position
